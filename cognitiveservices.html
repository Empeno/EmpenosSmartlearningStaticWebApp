Cognitive Services

Azure Cognitive Services er cloud baserede tjenester.
Der er altså tale om flere services med forskelligt fokus indenfor kunstig intelligens. Disse services fungerer via REST API'er og SDK'er.
API'er og SDK'er hjælper dig med at opbygge kognitiv intelligens i dine applikationer. Du kan tilføje kognitive funktioner til dine applikationer uden at have erfaring med kunstig intelligens (AI) og det kræver ikke flere års erfaring i software udvikling, da den største kompleksitet i selve softwaren ligger i API'er og SDK'er, og dermed er abstraheret væk fra klienten.
Azure Cognitive Services omfatter forskellige AI-tjenester, der giver dig mulighed for at opbygge kognitive løsninger, der kan se, høre, tale, forstå og endda træffe beslutninger.
