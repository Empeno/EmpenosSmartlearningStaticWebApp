What is Azure Static Web Apps?

Azure is the name of the Microsoft cloud platform. Azure contains a lot of services for Web, Database, Storage, Artificial Intelligence and more. Azure Static Web Apps is one of these cloud based services from Microsoft. With Azure Static Web Apps you can quickly setup automatic builds and deploys, so that your code is automatically released to the public available website, shortly after you have comitted the code to GitHub. Yoy are able to deploy full stack web apps to Azure from a GitHub repository.
